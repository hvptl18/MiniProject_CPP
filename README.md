# 🎮 Rock Paper Scissors Game (C++)

## 📌 Description
This is a simple console-based Rock Paper Scissors game developed in C++.  
The user plays against the computer, and the winner is decided based on standard game rules.

---

## ⚙️ Features
- User vs Computer gameplay  
- Random computer choice  
- Win / Lose / Draw result  
- Option to play multiple rounds  

---

## 🧠 Concepts Used
- Functions  
- Conditional statements (if-else)  
- Loops (do-while)  
- Random number generation (`rand()`)  
- Object-Oriented Programming (Class)

---

## 🎮 How to Play
1. Run the program  
2. Choose:
   - 1 for Rock  
   - 2 for Paper  
   - 3 for Scissors  
3. Computer will generate its choice  
4. Result will be displayed  
5. Choose to play again or exit
